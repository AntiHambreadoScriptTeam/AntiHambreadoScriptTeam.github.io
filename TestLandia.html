    Así de simple

    Pegas el enlace. Nosotros seguimos la ruta.

    Primero revisamos la URL, luego se la pasamos a la API y al final te mostramos lo que devuelva. La consola acompaña la espera para que no parezca que la página se quedó trabada.

    1. 01

      Revisamos el enlace

      Comprobamos que empiece con http:// o https://. Si no, te avisamos antes de enviarlo.

    2. 02

      Se lo pasamos a la API

      La API recibe la URL y busca el destino. Si tarda más de 45 segundos, detenemos el intento para que puedas probar otra vez.

    3. 03

      Te damos el resultado

      Si la API encuentra el destino, podrás abrirlo desde aquí. Si devuelve texto o algo falla, también te lo diremos tal cual.

    Revisa que el enlace esté completo y vuelve a intentarlo un poco más tarde. La lista dice qué redes admite la API, pero no confirma que todos sus enlaces funcionen todo el tiempo.

    Antes de pegar un enlace

    La URL se comparte con la API externa para poder resolverla. No pegues enlaces con contraseñas, tokens ni información privada.
